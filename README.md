### Naxerion AI — Multi-Modal Drug Recommendation System
A drug recommendation engine that processes text, audio, and images simultaneously. Not just a classifier — it reasons across modalities.
- Fused **RoBERTa** (clinical text), **Wav2Vec2** (patient voice/emotion), and **CLIP** (drug image verification) through a **dynamic Softmax attention layer** that weights each modality by confidence. Hit ~90% cross-modal accuracy.
- Fine-tuned **Gemma** on clinical data using SFT + DPO with Chain-of-Thought self-distillation. Deployed with STT/TTS and real-time SSE streaming at **sub-1.5s end-to-end latency**.
- Added a **DQN with experience replay** that optimizes recommendations from patient feedback over time — **+7% accuracy, -12% adverse predictions**.
- Production API: Flask + sliding-window rate limiting + RBAC + XSS sanitization + HIPAA audit logging. Dockerized with PostgreSQL, Kubernetes, NVIDIA GPU targeting **<200ms inference**.
